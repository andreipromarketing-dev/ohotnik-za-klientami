# Охотник за B2B-клиентами

Парсер B2B-контрагентов с автоматическим обогащением данных.

## Что делает программа

Программа находит компании по нише и городу, парсит контакты и обогащает данные через AI:

1. **Поиск компаний** — через Google Maps (SearchApi.io) + DuckDuckGo как fallback
2. **Фильтрация агрегаторов** — двухуровневая система: домены/паттерны + контентный анализ
3. **Парсинг контактов** — телефон, email, соцсети (VK, Telegram, MAX)
4. **AI-обогащение** — поиск ЛПР, дополнительных контактов через Groq/UncloseAI/LM Studio
5. **VK API fallback** — поиск группы/контакта в VK если не найден на сайте

## Как запустить

### Windows

Запустите файл `start.bat` — откроется веб-интерфейс в браузере.

### Linux/Mac

```bash
chmod +x start.sh
./start.sh
```

Интерфейс доступен по адресу: `http://localhost:8501`

## Требования

- Python 3.10+
- Playwright + Chromium (устанавливается автоматически)
- **(опционально)** Groq API — бесплатный ключ для AI-обогащения

## Установка

```bash
pip install -r requirements.txt
playwright install chromium
```

## Настройка

1. Скопируйте `.env.example` в `.env`
2. Заполните ключи:

```env
# Для поиска сайтов (SearchApi.io)
SEARCHAPI_API_KEY=ваш_ключ

# Для поиска в VK
VK_TOKEN=ваш_токен_vk

# Для Groq AI (бесплатный)
GROQ_API_KEY=ваш_ключ_groq
```

### Где взять ключи

- **SearchApi.io**: https://www.searchapi.io/ — поиск сайтов компаний
- **VK Token**: https://vkhost.github.io/ — поиск групп/контактов в VK
- **Groq API**: https://console.groq.com/ — бесплатный AI (Llama 3.1)
- **LM Studio**: https://lmstudio.ai/ — локальный AI (бесплатно, без ключей)
- **UncloseAI**: бесплатный AI (без ключей, нужен VPN)

## Как использовать

1. Запустите `start.bat`
2. Выберите AI-провайдера (Groq по умолчанию)
3. Укажите нишу, город, количество контрагентов
4. Нажмите **"ШАГ 1. Поиск"** — начнётся поиск компаний
5. Нажмите **"ШАГ 2. Парсинг + AI"** — AI обогатит данные (телефоны, email, ЛПР, соцсети)
6. Скачайте результат в Excel

## Система фильтрации агрегаторов

### Уровень 1: Быстрый фильтр (до запроса)
- Блок-лист 72+ доменов (2GIS, Яндекс Карты, Avito, profi.ru и др.)
- Паттерны URL (`/firm/`, `/catalog/`, `/reviews`, длинные ID)
- Паттерны заголовков (отзывы, рейтинг, каталог)
- Punycode — только конкретные агрегаторы

### Уровень 2: Контентный анализ (после загрузки страницы)
- Система подсчёта: телефоны, текстовые сигналы, рейтинги, адреса, каталог-маркеры
- Сигналы оригинального сайта снижают скор
- Порог: score >= 4 = агрегатор
