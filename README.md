# Охотник за клиентами

Парсер базы контрагентов с автоматическим обогащением данных.

## Что делает программа

Программа создает Excel/CSV файл с компаниями и для каждой компании:

1. **Ищет сайт** — через Google Search (SearchApi.io)
2. **Парсит контакты** — телефон, email, соцсети (VK, Telegram, MAX)
3. **Если контактов нет** — использует AI (локальный LLM) для поиска ФИО директора/ЛПР на сайте

## Как запустить

### Windows

Просто запустите файл `start.bat` — откроется веб-интерфейс в браузере.

### Linux/Mac

```bash
chmod +x start.sh
./start.sh
```

Интерфейс доступен по адресу: `http://localhost:8501`

## Требования

- Python 3.10+
- **(опционально) LM Studio** — внешняя программа для локального AI. Скачайте с https://lmstudio.ai/, установите модель и запустите. Программа подключится к ней автоматически.

## Установка

```bash
pip install -r requirements.txt
```

## Настройка

1. Скопируйте `.env.example` в `.env`
2. Заполните необходимые ключи:

```env
# Обязательно для поиска сайтов
SEARCHAPI_API_KEY=ваш_ключ

# Опционально для поиска в VK
VK_TOKEN=ваш_токен_vk

# Для AI (опционально)
LM_STUDIO_URL=http://localhost:1234/v1
```

### Где взять ключи

- **SearchApi.io**: https://www.searchapi.io/ (нужен для поиска сайтов компаний)
- **VK Token**: https://vkhost.github.io/ (для поиска групп/контактов в VK)
- **LM Studio**: https://lmstudio.ai/ (для локального AI без облачных API)

## Как использовать

1. Запустите `start.bat`
2. В интерфейсе выберите: нишу, город, количество контрагентов
3. Нажмите "Запуск" — начнётся первичный поиск, результаты сохранятся в таблицу
4. Нажмите "Шаг 2" — AI (LM Studio) обогатит данные (ФИО директора/ЛПР/почта/соцсети - если есть на сайте)
5. Скачайте результат в Excel

Внизу интерфейса — лог операций (можно использовать для отладки/доработки в Claude Code, Cursor, OpenCode)

## Структура файлов

```
├── app.py              # Интерфейс Streamlit
├── config.py           # Настройки
├── enricher.py         # Парсинг и обогащение
├── search_providers.py # Поиск через SearchApi
├── lm_studio_client.py # AI-клиент (LM Studio)
├── start.bat           # Запуск (Windows)
└── requirements.txt    # Зависимости
```
